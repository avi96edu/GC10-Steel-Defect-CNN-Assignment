## Overview
This project classifies steel surface defect images from the GC10-DET dataset using custom CNN architectures.

## Dataset
- Dataset: GC10-DET
- Classes: 10
- Type: Grayscale steel surface defect images

## Models
- CustomCNN-Base
- CustomCNN-BN
- CustomCNN-Drop

## Best Submitted Model
**CustomCNN-BN**

## Best Result
- Accuracy: 66.86%
- Macro-F1: 0.5640

## Files
- `CNN_<StudentID>.ipynb` → main submission notebook
- `results/` → plots and outputs
- `extra_research/` → additional reliability experiments

## How to Run
Open the notebook and run cells from top to bottom.
